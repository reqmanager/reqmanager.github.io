# Screenshots Directory

This directory contains screenshots of the ReqMan application for display on the GitHub Pages site.

## Required Screenshots

Please add the following screenshots to this directory:

1. **dashboard.png** - Main dashboard showing project overview
2. **requirements.png** - Requirements management page with hierarchical view
3. **matrix.png** - Traceability matrix showing requirements to tests mapping
4. **requirement-detail.png** - Detailed view of a single requirement
5. **tests.png** - Test management page
6. **categories.png** - Categories and applicability management page

## Image Guidelines

- **Format**: PNG or JPG
- **Recommended size**: 1920x1080 pixels (16:9 aspect ratio)
- **File size**: Keep under 500KB per image for fast loading
- **Naming**: Use the exact filenames listed above

## Taking Screenshots

1. Start the ReqMan application: `cargo run --bin req_man`
2. Navigate to each page in your browser
3. Take screenshots using your browser's developer tools or a screenshot tool
4. Save them with the exact filenames listed above
5. Place them in this `screenshots/` directory

The webpage will automatically display these screenshots in a responsive gallery layout.
